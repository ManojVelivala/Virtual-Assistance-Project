# 🎙️ John - Intelligent Voice Assistant

John is a browser-based Virtual Assistant developed using HTML, CSS, and JavaScript. It leverages the Web Speech API to provide voice recognition and speech synthesis capabilities, allowing users to interact with the assistant through natural voice commands.

## 🚀 Features

### 🎤 Voice Recognition
- Listens to user commands using the Web Speech API.
- Converts speech into text in real time.

### 🔊 Voice Responses
- Responds using browser-based Speech Synthesis.
- Provides natural voice feedback for user interactions.

### 🤖 Smart Command Processing
John can perform various tasks including:

- Personalized greetings based on the current time of day.
- Opening popular websites:
  - YouTube
  - Google
  - Facebook
  - Instagram
- Performing Google searches.
- Displaying the current date and time.
- Launching utility applications such as Calculator (browser-supported environments).
- Answering common user queries.

### 🎨 Modern User Interface
- Responsive and clean design.
- Built using CSS Flexbox.
- Smooth user experience across supported devices.

## 🎯 Usage

1. Launch the application.
2. Click the **"Talk to Me"** button.
3. Allow microphone permissions.
4. Speak a command.

### Example Commands

```text

What is the time?
